Pliki z wbudowanym kodem QR

Obecna wersja DOSPRN potrafiła tworzyć kody QR z plików danych i drukować je jako część raportów.

Polecenia ucieczki mogą mieć format

ESC~7;ZZ;"qrcode raw text" - print a QR code from the raw text

ESC~8;ZZ;"filename.txt" - print the QR code from the file in UTF-8

ESC~9;ZZ;"filename.txt" - print a QR code from the file with the same code page as the main report

where ZZ - is a zoom factor
